Disruptor-消费模式简介(池化)

2021-03-01
Java
Disruptor , Java

并行模式(池化)

每个消费端有两个线程实例

disruptor.handleEventsWithWorkerPool(new A1Handler(),new A1Handler());
disruptor.handleEventsWithWorkerPool(new A2Handler(),new A2Handler());

结果示例

可以看到每次执行的线程是不一样的

串行模式（池化）

每个消费端有两个线程实例

disruptor.handleEventsWithWorkerPool(new A1Handler(),new A1Handler())
                .then(new A2Handler(),new A2Handler());
